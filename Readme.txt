# Quadiro Technologies Assessment

## Description

This project implements the required admin and user features for the Quadiro Technologies assessment.

## Features

- Admin login - http://localhost:5000/api/admin/login 

Admin credentials
    {
  "username": "admin_username",
  "password": "admin_password"
}

- CRUD operations for car records - http://localhost:5000/api/admin/cars
- Dashboard displaying car statistics - http://localhost:5000/api/user/cars
- User login - http://localhost:5000/api/user/login
